Homework . Трансформація

Css властивість translate дозволяє змінювати положення елемента

Основні властивості translate:

  • translate
  • rotate
  • scale
  • skew
  • matrix
  • perspective
  • translate3d

Основні правила

  • Обертаючи об'єкт, його система координат переміщується разом з ним
  • Переміщуючи об'єкт він рухається відносно власної системи координат (а не системи батьківського елемента)
  • Порядок запису значень може змінити результат
translate: -50px 70px
rotate: 200deg;
scale: 3
transform: skew(60deg, 30deg)
transform: translate(30px, 2px) rotate(-20deg) scale(0.9, 2) або transform: matrix(1.272, -2.121, 1.272, 2.121, 30, 2);

Трансформація 3d

  • perspective - задає відстань від площини об'єкта, або, іншими словами, силу перспективи. Чим більше значення, тим далі ви знаходитесь від об'єкта; чим менше значення, тим більш помітною буде перспектива
  • perspective origin - визначає позицію, з якої ви "дивитеся" на об'єкт. Якщо початок перспективи розташований по центру (за замовчуванням), а об'єкт зміщений праворуч, то буде здаватися, що ви дивитеся на нього зліва (і навпаки).
  • Крім того, ви можете залишити об'єкт по центру і перемістити початок перспективи. Коли початок координат встановлений збоку, здається, що ви "дивитеся" на об'єкт з цього боку. Чим більше значення, тим далі вбік він буде виглядати.
transform: rotate3d(100, 1, 1, 50deg)
transform: translate3d(20px, 20px, 40px) rotate3d(1, 1, 1, 50deg)
transform: translate3d(20px, 20px, 40px) rotate3d(1, 1, 1, 50deg)
transform: translate3d(20px, 20px, 40px) rotate3d(1, 1, 1, 50deg)
